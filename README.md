# Umlaut

German umlauts on non-German keyboards are often troublesome. 
**Umlaut** is a very simple solution to quickly type umlauts.

## Instructions

You need | You press
-------- | ---------
ä | AltGr + a
Ä | AltGr + Shift + a
ö | AltGr + o
Ö | AltGr + Shift + o
ü | AltGr + u
Ü | AltGr + Shift + u
ß | AltGr + s
